# Automated Sat Hunter
Automated script to hunt for rare sats.

## Requirements
- NodeJS
- Bitcoin Core

## Setup
Install
```agsl
git clone https://github.com/deezy-inc/sat-hunter.git
cd sat-hunter
npm install
```
Setup ENV variables
```agsl
cp .env.example .env
// Edit .env to match your setup.
```

## Usage
```agsl
npm start
```
